# Como usamos Ref y para que sirve


Ref tienes varios usos y caracteristicas interesantes pero para que sirve

1. Ref te da acceso directo a un elemento del DOM
2. Puedes leer y modificar sus propiedades
3. Esta referencia perciste entre render
4. es un hook asi que sigue sus reglas


Aspectos importantes y que lo hacen unicos
1. No cause rerenders, Cuando nosotros usamos useState y cambia su valor este si causa rerender para modificar su valor pero ref no lo hace aunque modifiques su valor

1. Usa su propiedad current para tener informacion de los elementos que referencias

## Como usar Ref

1. Define ref

``` js
const countRef = useRef(0)

```

Como te puedes dar cuenta es una constante ahora ya creado se lo asignas a un elemento por ejemplo a un botton

``` js
// asignamos la constante a este elemtno
<button ref={countRef}>Click me</button>

```

Ahora si usamos la propiedad current del elemento podremos acceder a todas las propiedades de button.

Pero como te puedes dar cuenta nosotros definimos a ref con un valor 0 entonces ts lo infiere como un valor, pero nosotros lo asignamos como a un button como hacemos que ts ahora entienda que debe deter las propiedades de un button

``` js
// creamos elemento con el tipado de un button
const countRef = useRef<HTMLButtonElement>(null)

// lo asignamos

<button ref={countRef}>Click me</button>

// ahora lo usamos de esta forma accediendo a propiedades que tiene el boton    
countRef.current.value

```

Ejemplos de tipos de elementos 

``` js
// Botón
const buttonRef = useRef<HTMLButtonElement>(null);

// Input
const inputRef = useRef<HTMLInputElement>(null);

// Div
const divRef = useRef<HTMLDivElement>(null);

// Textarea
const textareaRef = useRef<HTMLTextAreaElement>(null);

// Canvas
const canvasRef = useRef<HTMLCanvasElement>(null);

```

Con ref podemos usar todas las propiedades, valores, metodos, todo

```js

const buttonRef = useRef(null);

// Puedes usar métodos como:
buttonRef.current.click();      // Simular un click
buttonRef.current.focus();      // Dar foco
buttonRef.current.blur();       // Quitar foco
buttonRef.current.scrollIntoView(); // Scroll hacia el elemento

// Y acceder a propiedades como:
buttonRef.current.disabled = true;
buttonRef.current.textContent = "Nuevo texto";

```
