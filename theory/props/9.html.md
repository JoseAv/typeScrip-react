# Props para elementos de html

Como ya vimos en la seccion de css que existe un React.CSSProperties que nos ayuda a ya tener tipado todas las propiedades css tambien existe para los elementos de html.

Aunque este tema es mas variante ya que sabemos que dependiendo de un componente puede tener uno o otros elementos 

Ejemplo;

- <a> tiene href
- <button> tiene onClick
- <select> tiene opcion y values
- <input> tiene types, disable, etc...

Para todos estos elementos se tiene la propiedad ComponentProps.

## ComponentProps

Esto es lo que usamos para indicar que vamos a usar las propiedades de un elemento html como funciona

``` js
// ComponentProps y el elemento que queremos las props
type LinkProps = ComponentProps<a | button | input>


```

Ejemplo completo

``` js

import { ComponentProps } from "react";

type ButtonProps = ComponentProps<"button">;

// aqui desustructuramos los tipos del boton
function Button(props: ButtonProps) {
    return <button {...props}>{} </button>;
}
    function Page() {
    return(
        <div>
           <Button />
        </div> 
)
}

```

### ComponentPropswitoutRef

Esta es una recomendacion que nos hace react ya que para poder interactuar directamente con el dom react nos remienda usar ref para la referencia en el dom, pero en este caso no lo estamos usando entonces estamos utilizando ComponentPropswitoutRef pero tambien podriamos usar la prop ComponentPropswittRef cuando usemos ref

Ejemplo :


``` js

import { ComponentPropsWithoutRef } from "react";

type ButtonProps = ComponentPropsWithoutRef<"button">;

// aqui desustructuramos los tipos del boton
function Button({onClick, ...rest}: ButtonProps) {
    return <button onClick={onClick} {...rest}>Tocame </button>;
}
    function Page() {
    return(
        <div>
           <Button />
        </div> 
)
}

```
