# Como tipar propiedades css

en react nosotros podemos pasar props para definir estilos, esto es una practica comun y lo podemos hacer de esta forma

``` ts
interface css {
    color: 'red' | 'blue'
    text: string
}

const Button = ({color,text}: css){
    return (
        <button style={{color,text}}></button>
    )

}

```

Aqui estamos pasando las propiedades para que style las pueda usar pero esto aunque este bien es bastante dificil de escalar ya que si queremos extender o cambiar otras propiedades tendremos que ir agregando valores y valores pero ya existe una propiedad para esto

# React.CSSProperties

React.CSSProperties es una propiedad que ya nos trae todos las propiedades de css, color,backgroud,text,font, etc...

ejemplo :

``` ts
type ButtonProps = {
    style: React.CSSProperties;
};
 I
function Button({ style }: ButtonProps) {
    return (
    <button
    style={style}>
        test
    </button>
);
}

function Page() {
    return (
    <div>
        <Button style={{
            backgroundColor: "gold",
            color: "black",
            fontSize: "160px",
            padding: "100px",
            }} />
    </div>
    )
}


```
