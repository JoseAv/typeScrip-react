# Arrays en JS

Para indicar a TS que algo es un array usamos la siguiente notacion

``` js 
    // Arreglo tipo number
    type array = number[]

    // Cualquier tipo
    type array = []

    // Union Types
    type array = number[] | string[]

```

# Tuplas

Las tuplas son arreglos de un espacio limitado o permitido y esta es una funcionalidad especial que solo tiene ts y se especifica de esta forma


``` ts
    // Arreglo tipo number
    type tupla = [number, number] // -> es un arreglo de 2 posiciones y las 2 posiciones tiene que se un numero

    // Cualquier tipo
    type tupla = [string,string,number] // -> una tupla conbinada

    // Union Types
    type tupla = [string, string, string?, number?] // -> tambien puede tener algunas props opcionales
```
