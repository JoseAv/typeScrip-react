# Union Types

este caracter ***|*** significa union y funciona para unificar cosas por ejemplo

``` js
type ButtonProps = {
subtitle?: string; 
text: string;
color?: string;
backgruondColor?: "red" | "blue" | "green" | "yellow";
sizes: 30 | 40 | 50;
};

// Aqui backGroundColor solo puede ser ese valor
// Si intentamos pasarle otro como 'Gray' daria error ya que no lo acepta

// Podemos crear tipos solo de los colores y compartilos

type colors = "red" | "blue" | "green" | "yellow" | "pink"
type sizes = 30 | 40 | 50

type ButtonProps = {
subtitle?: string; 
text: string;
color?: string;
backgruondColor?: colors;
sizes: sizes;
};

```
Hasta aqui lo hemos hecho solo con tipos primitivos pero tambien se podria unificar objetos o cualquier otro tipo como por ejemplo:


``` js
type person = {name:string,lastname:string}
type fondos = {total:number,debe:number}
/// ahora los unimos
type informationPerson = person | fondos

// Resultado // asi se veria informationPerson

type I = {
    name:string,
    lastname:string,
    total: number,
    debe:number
}

```
