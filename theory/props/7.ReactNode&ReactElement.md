# ReactNode & ReactElement


#### ReactNode

ReactNode son todos los elementos que puede devolver y son permitidos en react Ejemplo


``` js
interface prueba {
    Comp: ()=> ReactNode
}

const Comp = ()=>{
    return(
        <>
        <h1>Hola</h1>
        </>
    )
}
```

Ahora si queremos que otro componente lo tome para el mostrarlo lo podemos hacer de esta forma

Ejemplo


``` js

const Comp = ()=>{
    return(
        <>
        <h1>Hola</h1>
        </>
    )
}

const Hola = ({children}: {children: ReactNode})=>{
    return(
        {children}
    )
}

<Hola><Comp/></Hola>

```

como podemos ver children es el objeto especial que nos ayuda a indicar que el componente esta envolviendo a otro entonces esta tomando ese componente y lo esta devolviendo, pero a TS necesitamos decirle que tipo es la prop children

#### React Element

Nosotros tambien podemos utilizar indicando especificamente que solo vamos a devolver un elemento jsx asi:

```js
interface prueba {
    children: JSX.Element 
}

```

esto va a indicar que tipo de elemento devuelve children aqui especificamos que sera si o si un elemento de JSX pero solo puede ser 1 element

###### Multiples elementos

para indicar que queremos pasar varios elementos JSX lo hacemos de esta forma

``` js
interface prueba {
    children: JSX.Element[]
}

```
