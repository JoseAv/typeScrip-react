## Props por defecto

Hasta el momento hemos visto como pasar props y hacerlas opcionales pero tambien podemos asignar valores por defecto si cierto valor no es pasado y se hace de seta forma


```ts
// esto es valido y es la forma  que lo hacemos
type typeTitle = {
    title:string
}

const Button =({title}: typeTitle)=>{

    return (
        <h1>{title}</h1>
    )

}

// que pasa si el valor es opcional 

type typeTitle = {
    title?:string
}
const Button =({title}: typeTitle)=>{

    return (
        <h1>{title}</h1> // este valor puede no ser nada
    )

}


// Resultado

// h1 -> No mostraria nada pero no daria error

// Como arreglamos esto


const Button =({title= 'Hola'})=>{

    return (
        <h1>{title}</h1> // Si no le pasamos nada el texto dira Hola
    )

}


```

Ahora esta forma tiene una peculiarida que es que podemos no tiparlo ya que al hacer la asignacion de string entonces el ya sabe que text es string como cuando declaramos una variable.
