# Tipado de funciones

Nosotros podemos pasar funciones a nuestros componentes y para tiparlas lo hacemos de esta forma

``` js
// Una funcion que no devuelve nada
interface typeFunction {
    prueba : ()=> void
}

// Una funcion con parametros
interface typeFunction {
    prueba : (text: hola)=> void
}

// Una funcion generica
interface typeFunction<T> {
    prueba : (text: T)=> T
}

// Funcion que retorna un componente
interface typeFunction {
    prueba : ()=> ReactNode;
}

```
