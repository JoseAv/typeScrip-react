# Utility Type

Son funciones que nos permite manipular types o interfaces para tener mas control sobre ellas, cada una tiene una funcion diferente las veremos a continuacion.

# Omit
Esta sirve para indicar que queremos todo el objeto excepto algun valor que vamos a indicar por ejemplo: usuarios que no queremos el password lo podemos excluir de esta forma

``` ts
type user = {name:string
            age: number
            password:string
            }

// aqui usamos omit, primero indicamos el objeto y despues la propiedad a sacar
type userWithoutPassword = Omit<user,'password'>
// este seria el nuevo type

type result = { name?:string
                age?: number
              }


```



# Partial
Esta volvera todos los campos del objeto opcionales por si estamos haciendo una edicion entonces el usuario puede mandar cualquier campo como opcional ya que puede o no puede enviar informacion

```ts
type user = {name:string
            age: number
            password:string
            }

type userOptional = Partial<user>

// este seria el nuevo type

type result = { name?:string
                age?: number
                password?:string
              }

```


# Pick
Esta utilidad va  a tomar aquellas propiedades que le indiquemos es ultil si solo queremos tomar algunas partes


``` ts

type user = {name:string
            age: number
            password:string
            }

// Tomamos el objeto y despues las propiedades a tomar
type validationUser = Pick<user, 'age' | 'password'>

```

# Exclude
Exclude es lo mismo que Omit pero este sirve para los UnionType que son la union de varios valores con el operador  -> |

```ts
type names = 'Jose' | 'Miguel' | 'Alberto'

// Tenemos un tipo con nombres prohibidos por ejemplo

type warningNames = Exclude<names, 'Jose'>

// Resultado

'Miguel' | 'Alberto'


```

pero y si lo queremos para valores de un  objeto literal

``` ts
type toys = {name:'Buzz'; price:number} | {name:'Pink'; price:number}

// queremos quitarle el precio ya no esta a la venta
type toysWithoutPrice = Exclude<toys, {name:'Buzz'}>

// Resultado

type result = {name:'Pink'; price:number}


```
